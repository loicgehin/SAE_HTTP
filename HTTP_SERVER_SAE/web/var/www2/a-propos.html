À Propos de ce projet

Ce site statique (HTML/CSS) a été réalisé dans le cadre du projet DIW (Développement d'interfaces Web) de la SAÉ 1.06.

Thème

Le thème personnel choisi est "Les musiciens que j'aime", avec 3 rubriques : A Tribe Called Quest, Pink Floyd, et Stevie Wonder.

Maquettes

Les maquettes ont été dessinées à la main pour définir la structure des pages.

Justifications

flexbox a été utilisé à plusieurs endroits :

  • Dans main pour centrer les boîtes et aligner en vertical partout.
  • Dans .musiciens-cases sur la page d'accueil, pour centrer le titre au milieu des cases.
  • Dans .a-liens pour centrer les 2 cases proposants des articles

Je n'ai pas eu besoin de grid

Conclusion

Les dificultés principales ont été pour le format des images, au début je voulais coller 3 images différentes mais j'ai due les compilés pour les mettres en 1 format, j'ai aussi eu des difficulté car je voulait les mettres dans le html a la base mais au final il fallait le mettre dans le css en background image. J'ai aussi eu des difficulté pour la création des classe en css, j'ai due tous réaprendre mais c'était finalement assez facile a utilisé. En derniére difficulté, l'envergure des recherches a faire, j'ai donc fais quelque chose de plus concis et correspondant moins a ce que j'aurai voulu.
